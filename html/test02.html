1. html문서를 구성하는 필수태그들

   <html> ----- 문서의 시작을 선언
    <head> ----- 머리말 시작을 선언
     <title> ----- 제목 시작을 선언
     </title> ----- 제목 끝을 선언
    </head> ----- 머리말 끝을 선언
    <body> ----- 본문 시작을 선언

    </body> ----- 본문 끝을 선언
   </html> ----- 문서의 끝을 선언

2. 글자를 바꾸는 태그들

bold = <b> b 볼드체
bold = <strong> strong (중요하다는 의미로)볼드체
italic = <i> i 이텔릭체
italic = <em> em (강조한다는 의미로)이텔릭체
<small> = small 글자 크기를 작게
<sup> = sup 윗첨자 (ex) 30cm3, 100km2
<sub> = sub 아랫첨자 (ex) H2O, O3, CO2
<mark> = mark 형광펜 효과

markup O,X

1. bold mark = O
2. bold mark = X
3. bold mark = X
4. bold mark = O
